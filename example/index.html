Instructions

Basic

  • Alt + h hides 'One'
  • Alt + o shows 'One'
  • Alt + j hides/shows 'Two'
  • Alt + k hides/shows 'Three'
  • Esc hides all
  • Ctrl + m shows all

Subviews (view console debugger)

  • Alt+1 creates a sub-view
  • Alt+2 fires a non-global event on a sub-view
  • Alt+3 fires a global event on a sub-view
  • Alt+4 deletes the sub-view
